# Development

## Penumbra

Clone the penumbra repository to setup the development environment

```shell
$> git clone https://github.com/penumbra-zone/penumbra
$> cd penumbra
$> git checkout v1.4.0
```

Setup the development environment

```shell
$> nix develop
$> just dev
```
